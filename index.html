Project on Console

Visual Studio Code is a free source-code editor made by Microsoft for Windows, Linux and macOS.
To setup Visual Studio Code:
*First download VS Code from https://code.visualstudio.com/Download
*Open and add extensions to it like Beautify, ESLint, Live Server etc.
*Now create a new folder in it.
*In the same folder create a file with extensions like .html, .css, .js etc. to work in it.

Console application (app) is a program developed in Visual Studio which accepts input parameter, calls the required service, runs business logic and sends output to the console, this console is known as the command prompt.
To open console scree:
*Right click on the output screen.
*Click on 'Inspect'.
*Select 'Console' tab.
To print anything in console screen:
*Inside the script tag add the function called console.log().
*Inside the () give "" and write something you want to print in the console screen
*Don't forget to give the ; after you finish the line.
